General rules

  1. You will automatically be drawn the first card
  2. The deck contains 52 cards. When you run out, the game will restart
  3. You need 5 correct guesses in a row to win the game
  4. When you guess wrong, the counter of correct guesses resets
  5. If the value of the new card is the same as the previous, it will count as an incorrect guess
  6. Your stats will be stored when you win the game or run out of cards
